
Щоб завантажити програму на Arduino та скомпілювати її, дотримуйся таких кроків:

1. Завантаження та встановлення Arduino IDE:Перейдіть на офіційний сайт Arduino та завантажте Arduino IDE для вашої операційної системи (Windows, macOS, Linux).
Встановіть Arduino IDE, слідуючи інструкціям інсталятора.

2. Підключення Arduino до комп'ютера: Підключіть вашу плату Arduino до комп'ютера за допомогою USB-кабелю.


3. Запуск Arduino IDE: Запустіть Arduino IDE, натиснувши на його іконку на робочому столі або через меню програм.


4.  Вибір плати Arduino: У меню "Tools" (Інструменти) виберіть "Board" (Плата) і оберіть свою модель плати Arduino зі списку (наприклад, Arduino Uno, Arduino Nano тощо).


5. Вибір порту: У тому ж меню "Tools" виберіть "Port" (Порт) і оберіть COM-порт, до якого підключена ваша плата. Зазвичай це щось на кшталт "COM3" на Windows або "/dev/tty.usbmodemXXXX" на macOS/Linux.
Створення або відкриття скетчу:

6. Компілляція програми: Натисніть кнопку "Verify" (Перевірити) у верхній частині вікна IDE (значок з галочкою). Це запустить процес компіляції вашого коду, щоб перевірити наявність помилок.

7. Завантаження програми на плату: Якщо компіляція пройшла успішно, натисніть кнопку "Upload" (Завантажити) (значок зі стрілкою вправо). Це скомпілює та завантажить програму на вашу плату Arduino.
